## 🚀 Características

| Característica | Descripción |
|----------------|-------------|
| 🎨 **Diseño Futurista** | Glassmorphism, efectos neon, animaciones fluidas |
| 🔐 **Autenticación** | Registro/Login con persistencia en localStorage |
| 🛒 **Carrito de Compras** | Persistente por usuario, actualización en tiempo real |
| 💳 **Pasarela de Pago** | Simulación con tarjeta/efectivo, validación de datos |
| 🎫 **Generación de Tickets** | Factura detallada con opción de impresión |
| 🔍 **Buscador** | Filtrado de productos en tiempo real |
| 📱 **Responsive** | Adaptable a móviles, tablets y desktop |
| 💾 **Persistencia** | Datos guardados localmente |

## 🎯 Funcionalidades

### 🔐 Autenticación
- ✅ Registro de nuevos usuarios
- ✅ Inicio de sesión seguro
- ✅ Cierre de sesión
- ✅ Cuentas demo preconfiguradas
- ✅ Menú de usuario con opciones

### 🛒 Carrito de Compras
- ✅ Añadir/eliminar productos
- ✅ Actualizar cantidades
- ✅ Cálculo automático (subtotal, IVA 16%, total)
- ✅ Persistencia por usuario

### 💳 Checkout
- ✅ Métodos de pago: Efectivo / Tarjeta
- ✅ Validación de tarjeta (número, fecha, CVV)
- ✅ Generación de ticket/factura
- ✅ Descarga/impresión de comprobante

### 🎨 UI/UX
- ✅ Efectos glassmorphism
- ✅ Animaciones suaves
- ✅ Hover effects futuristas
- ✅ Notificaciones visuales

## 🛠️ Tecnologías

| Tecnología | Uso |
|------------|-----|
| **React 18** | Biblioteca principal para UI |
| **Vite** | Build tool y servidor de desarrollo |
| **CSS3** | Animaciones, Flexbox, Grid, Glassmorphism |
| **localStorage** | Persistencia de datos |

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/TU-USUARIO/watchdive-megamart.git

# Entrar al directorio
cd watchdive-megamart

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
